# Atividade prática individual - Tópico 3
## Nível realizado
Nível 3 - Avançado

## Objetivo
Criar, publicar, validar e documentar a infraestrutura necessária para hospedar uma aplicação WordPress com Nginx em ambiente Linux.

## Ambiente utilizado
Linux no Browser (Killercoda / Ubuntu 24.04)

## Rota de publicação
WordPress com Nginx (LEMP)

## Ficheiros criados
- comandos.txt
- publicacao.md
- validacao.md
- README.md

## URLs testados
`http://localhost`

## Evidências produzidas
Mapeamento técnico estrutural e validação teórica da Stack LEMP.

## Dificuldades encontradas
Isolamento de portas da sandbox do Killercoda que impede o redirecionamento de portas HTTP padrão para fora do terminal.
